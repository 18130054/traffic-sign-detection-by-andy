How to run
1. pip install tensorflow-cpu==1.15 keras==2.3.1 scikit-learn==0.22.2.post1 opencv-python
2. creating file model/cnn_model.h5
3. Download data set GTSDB save to my_data_circles folder, only take circle traffic
4. chcp 65001 (if facing error "UnicodeEncodeError: 'charmap' codec can't encode character '\u1ed5' in position 1: character maps to <undefined>")
5. python tran_cnn.py
